# Weather App using Open Weather API and flutter Bloc pattern

This is a complete flutter app which has Open Weather API integration. The complete app is built using Bloc Pattern

## Features
* Splash Screen
* App Icon
* API integration
* Flutter BLOC Pattern
* 2 Screen App
